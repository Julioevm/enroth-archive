# **App Name**: Enroth Archive

## Core Features:

- Interactive Area Maps: Display interactive maps of Might & Magic 6 areas, allowing users to explore and discover locations. Users will click locations for relevant information.
- Area Quick Access Grid: Display a grid of area map images on the homepage for quick access to interactive area maps.
- Dungeon Pages: Dedicated pages for each dungeon with detailed information and a map of the dungeon.
- Quest Database: A database of quests, sortable by area or dungeon. Include quest descriptions and relevant details.
- Skill Directory: A searchable directory of skills, including skill descriptions, trainers, and training locations.
- Location information with AI tools: When a user clicks on a location on a map, use an AI tool to determine relevant quests, skills and monsters to output as potential results.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5), evoking a sense of mystery and adventure, aligned with the Might & Magic theme.
- Background color: Light Gray (#F0F2F5), provides a clean and unobtrusive backdrop for the content.
- Accent color: Gold (#FFC107), used for highlights and interactive elements to signify important information and actions.
- Headline font: 'Space Grotesk', sans-serif for a clean, computerized and easily readable.
- Body font: 'Inter', sans-serif that complements 'Space Grotesk'; readability across devices.
- Use clear and consistent icons to represent different locations, quests, skills, and trainers.
- Use a grid-based layout for the homepage area map grid.
- Subtle animations on hover for map grid and location markers.
